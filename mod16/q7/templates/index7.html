MVT Architecture

Interactive example of Django architecture in one page. Dynamically rendered data on cards with clear Model-View-Template explanation.

Model

Defines doctor data fields and database schema in models.py. In this demo, the list is in the view for quick preview.

View

q7/views.py builds doctors list and renders template with render(request, 'index7.html', context).

Template

This file loops doctors and renders cards. It is your presentation layer.

{% for doctor in doctors %}
{{ doctor.initials }}
{{ doctor.name }}
{{ doctor.speciality }}
{{ doctor.bio }}
{% for link in doctor.links %} {{ link.emoji }} {% endfor %}
{% endfor %}
